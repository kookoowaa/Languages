# 비교연산자

## ===

- 파이썬에 `==`가 있다면, JS 에는 `===`이 있음

- JS에도 `==`이 있지만, 성격이 일부 다르며 `===` 사용을 강력히 권장;

  ``` javascript
  alert(1=='1')		//true
  alert(1==='1')		//false
  ```

## !==

- 위와 마찬가지로 `!=`와 `!==` 또한 성격이 일부 다르며, `!==` 사용을 권장

  ```javascript
  alert(1!='1')		//false
  alert(1!=='1')		//true
  ```

## &&, ||

- `&&`은 파이썬의 `and`와 같은 역할을 수행
- `||`은 파이썬의 `or`과 같은 역할을 수행
